# Steward's Initiative

A Crusader Kings III mod that automatically upgrades buildings on your behalf — across your own domain, and (optionally) subsidized across your vassals' holdings realm-wide — so you don't have to click through the building UI for every holding every session.

Configured entirely in-game via a decision in the decisions panel, which opens a settings menu. You control:

- Independent on/off toggles for domain upgrades and vassal subsidies
- A gold/treasury reserve floor for each (the mod never spends below it)
- Your own priority order across 6 building categories: Tax/Economy, Development Growth, Levy, Men-at-arms, Fortification, and Specials

Targets CK3 1.19.x ("Scribe"). See [`docs/superpowers/specs/2026-07-03-stewards-initiative-design.md`](docs/superpowers/specs/2026-07-03-stewards-initiative-design.md) for the full design spec.
